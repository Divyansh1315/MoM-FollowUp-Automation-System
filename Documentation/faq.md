# FAQ

## Q1. Why didn’t I receive a reminder?
- Task might be marked "Closed"
- Email might be incorrect
- Target date might be invalid

## Q2. What happens if I forget to update status?
- System keeps sending reminders

## Q3. Can I send a reminder manually?
Yes:
- Set ManualReminderTrigger = Yes

## Q4. What if Excel file is locked?
- Flow will retry automatically

## Q5. Can this system scale?
Yes:
- Designed for future migration to Dataverse

## Q7. Can I stop reminders temporarily?
- Set Status = Closed (temporary workaround)
